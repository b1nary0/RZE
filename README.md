**Setup:**
Once you have the repo locally:
1) Run GenerateSolutions.bat
2) Build solution
3) Right click SourceAssetBurner project -> Debug -> Start new instance
4) Run AssetCpy.bat
5) Start Editor project or Game project (Game is default)
